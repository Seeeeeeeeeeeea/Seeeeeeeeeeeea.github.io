科研与项目经历

面部属性可编辑的说话头视频生成算法研究

2025.03 - 2025.09

第四作者，中科院 SCI 一区在投

该研究面向数字人形象个性化定制，构建属性可编辑的音频驱动说话头视频生成框架。基于属性标注人脸数据训练语义潜空间线性分类器， 将其权重建模为可解释的面部属性编辑方向；推理时对参考人像语义潜变量进行属性方向调制，并作为语义引导条件参与视频去噪生成。

个人贡献： 负责部分对比实验的实施，完成定性分析与可视化展示，组织并分析用户调研结果；主导论文撰写与投稿工作。

音频-姿态解耦驱动的半身数字人视频生成算法研究

2025.09 至今

该研究面向数字媒体内容制作中的半身数字人视频生成场景，针对音频驱动面部动态时易受关键点条件干扰，以及生成嘴部动态易受参考图像初始嘴部姿态牵制的问题， 提出身份、语音动态与姿态动作的解耦控制机制。方法以参考图像建模人物身份与外观特征，以音频条件驱动口型及面部表情动态， 以稀疏关键点引导头部姿态与手势运动，并通过条件丢弃以平衡多条件对生成的控制。

个人贡献： 独立开展该项目研究，完成说话人驱动条件设计、扩散模型架构搭建及掩码音频交叉注意力核心模块实现； 负责大规模头部/半身说话人视频数据的收集、清洗与预处理，搭建多阶段训练流程，并完成模型训练、调优、对比实验与消融实验。

华为昇思大模型 2025 创新训练营（西北区）结营作品

2025.09

在华为云昇腾算力平台上，面向校园新生问答场景对 Qwen2.5-7B 模型进行微调。调用 DeepSeek-V3 API 合成数据集， 采用 LoRA 参数高效微调方法完成模型训练，并基于 Gradio 搭建 Web 交互平台，实现从数据构建、模型微调到系统部署的完整闭环。
